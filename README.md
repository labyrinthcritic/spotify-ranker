## Development

To run the dev server:
1. `cd app`
2. `npm install`
3. `npm run dev`

## Building

### Frontend
1. `cd app`
2. `npm run build`
The build results will be in `dist/`.

### Backend
The backend is a python module. Ensure that `secrets.json` is present in the root and run `python server`.
